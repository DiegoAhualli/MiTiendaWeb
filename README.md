# Busticaria

## Autor
Diego Ahualli

## Descripción
Busticaria es una aplicación web de ecommerce desarrollada como trabajo práctico para la materia Aplicaciones Web 1.

La aplicación simula una tienda online con distintas secciones y navegación entre páginas, aplicando buenas prácticas de estructura y estilos web.

## Funcionalidades implementadas

### Primera etapa
- Estructura base del sitio
- Página de inicio (Home)
- Página de login
- Página de registro
- Páginas de categorías de productos

### Segunda etapa
- Aplicación de estilos con Bootstrap y CSS propio
- Navbar responsive reutilizable mediante componente JavaScript
- Paleta de colores personalizada
- Diseño de login y registro centrado en la experiencia de usuario
- Implementación de cards de productos (prototipo)
- Mejora visual general del sitio (hero, categorías, botones, etc.)

## Tecnologías utilizadas
- HTML5
- CSS3
- Bootstrap 5
- JavaScript (uso básico para componentes reutilizables)

## Estado del proyecto
Segunda entrega finalizada – interfaz estilizada y navegación funcional
