# Roadmap de Desarrollo

## Etapa 1 – Estructura inicial del proyecto

En la primera etapa se desarrolló la estructura base de la aplicación web ecommerce.

### Objetivos principales

* Crear la estructura general del sitio.
* Implementar navegación entre páginas.
* Organizar carpetas y archivos.
* Definir identidad visual inicial.

### Desarrollo realizado

* Creación de páginas:

  * Home
  * Login
  * Registro
  * Categorías de productos
* Implementación inicial de HTML5.
* Definición de la estructura MVC básica.
* Creación del logo e identidad “Busticaria”.

---

## Etapa 2 – Diseño visual y experiencia de usuario

En esta etapa se trabajó principalmente sobre estilos, experiencia visual y reutilización de componentes.

### Mejoras implementadas

* Integración de Bootstrap 5.
* Implementación de CSS personalizado.
* Creación de paleta de colores propia.
* Diseño responsive adaptable.
* Implementación de cards de productos.
* Creación de hero principal.
* Mejora visual de formularios Login y Registro.
* Implementación de navbar dinámica reutilizable mediante JavaScript.

### Cambios y correcciones

* Corrección de rutas relativas.
* Reorganización de carpetas.
* Mejora del manejo de assets.
* Optimización visual general.

---

## Etapa 3 – Funcionalidad e integración con APIs

En esta etapa se incorporó funcionalidad dinámica y persistencia de datos.

### Desarrollo realizado

* Consumo de API DummyJSON.
* Renderizado dinámico de productos.
* Separación por categorías:

  * Tecnología
  * Ropa
  * Hogar
* Implementación de carrito de compras.
* Persistencia mediante LocalStorage y SessionStorage.
* Sistema de Login y Registro funcional.
* Validaciones de formularios.
* Manejo de usuario logueado.

### Mejoras técnicas

* Modularización del código JavaScript.
* Mejor organización MVC.
* Componentes reutilizables.
* Optimización de navegación.

---

## Etapa Final – Publicación y documentación

### Tareas realizadas

* Corrección final de rutas.
* Revisión de funcionalidades.
* Optimización visual final.
