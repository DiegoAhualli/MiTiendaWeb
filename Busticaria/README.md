# Busticaria

## Autor
Diego Ahualli

## Descripción
Busticaria es una aplicación web de ecommerce desarrollada como trabajo práctico para la materia Aplicaciones Web 1.

En la primera etapa se implementa la estructura base del sitio, incluyendo:

- Página de inicio (Home)
- Página de login
- Página de registro
- Páginas de categorías de productos

## Tecnologías utilizadas
- HTML5
- CSS básico

## Estado del proyecto
Primera entrega - estructura inicial de la aplicación
